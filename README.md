# Tic-Tac-Toe

### Description
A tic-tac-toe (noughts & crosses) game built with **React** and the **Framer Motion** animation library.

### Setup & Launch

#### Launch React App
1. Navigate to `cd tic-tac-toe`
2. Run `npm install`
3. Run `npm start`
4. Navigate to http://localhost:3000
